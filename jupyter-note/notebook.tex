\subsection{一. 引言介绍}\label{ux4e00.-ux5f15ux8a00ux4ecbux7ecd}}

强化学习 (Reinforcement learning)
是机器学习的一个子领域用于制定决策和运动自由度控制。强化学习主要研究在复杂未知的环境中，智体(agent)实现某个目标。强化学习最引人入胜的两个特点是

\begin{itemize}
\item
  \textbf{强化学习非常通用，可以用来解决需要作出一些列决策的所有问题：}例如，训练机器人跑步和弹跳，制定商品价格和库存管理，玩
  Atari 游戏和棋盘游戏等等。
\item
  \textbf{强化学习已经可以在许多复杂的环境中取得较好的实验结果：}例如
  Deep RL 的 Alpha Go等
\end{itemize}

\href{https://gym.openai.com/docs/}{Gym}
是一个研究和开发强化学习相关算法的仿真平台。

\begin{itemize}
\tightlist
\item
  无需智体先验知识；
\item
  兼容常见的数值运算库如 TensorFlow、Theano 等
\end{itemize}

\hypertarget{ux4e8c.-ux5f3aux5316ux5b66ux4e60ux7684ux57faux672cux6982ux5ff5}{%
\subsection{二.

强化学习也是机器学习中的一个重要分支。强化学习和监督学习的不同在
于，强化学习问题不需要给出``正确''策略作为监督信息，只需要给出策略的(延迟)回报，并通过调整策略来取得最大化的期望回报。
